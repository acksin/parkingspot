# ParkingSpot

ParkingSpot allows you to easily integrate AWS EC2 Spot Instances with other
services that you.

## Usage

```
parkingspot -h
parkingspot stats update
parkingspot ls
```

## Integrations

 - [ ] Terraform
   - [ ] Generate ASG
 - [ ] Packer
 - [ ] Kubernetes
 - [ ] Docker Machine
 - [ ] Fleet
